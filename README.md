# Stars Web

Web-based viewer and controller for the classic Stars! (Mare Crisium) game.

## Features

- Read Stars! binary save files and display game state in the browser.
- Interactive star map with planets, fleets, overlays, and detail panels.
- Submit turns and run the host from within the UI.
- **Inbox / turn messages**: view all diplomacy, battle, trade and event messages
  with filtering by year and message type.
- Production queue editing, waypoint management, and more.

See the [issues tracker](https://github.com/jakez-gh/stars-web/issues) for
complete backlog; key meta-issues: #204 (playable web game) and
# 205 (implement in-game report screens).
